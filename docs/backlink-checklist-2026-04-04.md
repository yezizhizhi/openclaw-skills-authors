# 外链清单（2026-04-04）

目标：为 `https://www.clawauthor.com` 建立第一批合规外链。

原则：

- 优先投放和 `AI tools / writing / creator workflow / startup launch` 相关的平台
- 优先选择真实收录、人工审核、仍在更新的平台
- 社区外链以“案例 / 经验 / 教程 / 反馈请求”为主，不做纯广告贴
- 目录站和社区站混合推进，不把策略全部押在目录站

---

## A. 提交目录站

### 1. Toolify
- 类型：AI 工具目录
- 提交入口：https://www.toolify.ai/submit
- 费用：付费
- 优先级：P1
- 备注：适合 AI 工具 / 导航站，页面明确写有提交入口和 dofollow SEO 卖点

### 2. Futurepedia
- 类型：AI 工具目录
- 提交入口：https://www.futurepedia.io/submit-tool
- 费用：付费
- 优先级：P1
- 备注：AI 目录头部平台，适合做品牌曝光和首批权威目录收录

### 3. TopAI.tools
- 类型：AI 工具目录
- 提交入口：https://topai.tools/submit
- 费用：付费
- 优先级：P1
- 备注：适合 AI 产品和 AI 导航站，提交页写明 24-48 小时审核

### 4. ToolList.ai
- 类型：AI 工具目录
- 提交入口：https://toollist.ai/submit-tool
- 费用：免费
- 优先级：P0
- 备注：免费入口，适合先拿首批目录站收录

### 5. AI Tools Inc
- 类型：AI 工具目录
- 提交入口：https://aitools.inc/submit
- 费用：免费排队 / 付费加速
- 优先级：P0
- 备注：有免费 listing，可先提交占位

### 6. Insidr.ai
- 类型：AI 工具目录
- 提交入口：https://www.insidr.ai/submit-tools/
- 费用：免费
- 优先级：P0
- 备注：支持提交 AI Tools，也支持后续 AI 新闻内容

### 7. ToolPilot
- 类型：AI 工具目录
- 提交入口：https://www.toolpilot.ai/pages/submit-your-ai-tool
- 费用：免费 / 付费
- 优先级：P1
- 备注：免费 listing 需要加回链，提交前要判断你是否接受这个条件

### 8. SaaSHub
- 类型：软件 / SaaS 目录
- 提交入口：https://www.saashub.com/services/submit
- 费用：免费
- 优先级：P0
- 备注：适合把 `clawauthor.com` 定位成创作者 AI 工具导航 / 资源产品

### 9. BetaList
- 类型：启动产品 / 早期产品目录
- 提交入口：https://betalist.com/submit
- 费用：普通提交通常免费，优先审核付费
- 优先级：P1
- 备注：适合把站点包装成“面向创作者的 OpenClaw Skills 导航产品”

### 10. Peerlist Launchpad
- 类型：产品发布 / Launch 平台
- 提交入口：https://peerlist.io/projects/2024/week/29
- 费用：免费
- 优先级：P1
- 备注：适合开发者与独立产品受众，偏 launch 型外链

### 11. Product Hunt
- 类型：产品发布平台
- 提交入口：登录后在 Product Hunt 点 Submit / New Product
- 参考说明：https://www.producthunt.com/launch/preparing-for-launch
- 费用：免费
- 优先级：P1
- 备注：更适合“正式 launch”，不要随便上；需要准备首评、素材、CTA

### 12. MicroLaunch
- 类型：启动产品 / distribution 平台
- 提交入口：https://microlaunch.net/premium/
- 费用：付费
- 优先级：P2
- 备注：更像 launch + 曝光服务，适合作为补充，不适合第一优先

### 13. Capterra
- 类型：软件目录 / 评测平台
- 提交入口：https://www.capterra.com/vendors/
- 费用：免费 listing
- 优先级：P2
- 备注：如果你后续把站点包装成可订阅的创作者工具平台，会更适合

### 14. GetApp
- 类型：软件目录 / 评测平台
- 提交入口 / 指南：https://www.getapp.com/listing_guidelines/
- 费用：通常通过 Gartner Digital Markets 流程处理
- 优先级：P2
- 备注：和 Capterra 属同一体系，适合后续补做

## B. 社区首批外链（合规）

### 15. Indie Hackers 产品页
- 类型：社区产品页
- 入口：注册后补齐个人资料和 Products 区
- 示例页面：https://www.indiehackers.com/kinder
- 优先级：P0
- 合规方式：把 `clawauthor.com` 作为产品页添加，不发广告式文案

### 16. Indie Hackers 案例帖 / Build in Public 帖
- 类型：社区帖子
- 社区参考：https://www.indiehackers.com/post/guide-how-to-do-community-based-marketing-ee5c766673
- 优先级：P0
- 合规方式：发“为什么我做了一个给创作者筛选 OpenClaw Skills 的站”或“从海量 Skill 到按场景导航”的经验贴，正文以经验为主，链接放自然位置

### 17. Hacker News / Show HN
- 类型：社区发布
- 入口说明：https://news.ycombinator.com/showhn.html
- 优先级：P1
- 合规方式：标题用 `Show HN:` 开头，正文讲你做了什么、为什么做，不要只有链接

### 18. DEV Community #showdev
- 类型：开发者社区展示帖
- 入口 / 规范：https://dev.to/t/showdev
- 优先级：P0
- 合规方式：写“Show DEV: I built a curated OpenClaw skills site for authors and creators”

### 19. Hashnode 复发文章（带 canonical）
- 类型：技术 / 产品文章社区
- canonical 官方说明：https://docs.hashnode.com/help-center/hashnode-editor/how-to-set-a-canonical-link
- 优先级：P0
- 合规方式：把站内一篇高质量文章转载到 Hashnode，并设置原文 canonical 到 `clawauthor.com`

### 20. LinkedIn 长文章
- 类型：职业社区文章
- 官方说明：https://www.linkedin.com/help/linkedin/answer/a522427
- 优先级：P0
- 合规方式：发“创作者如何选择 AI Skills”类文章，文章内自然引用站点

### 21. LinkedIn 创始人动态帖
- 类型：职业社区短帖
- 官方说明同上：https://www.linkedin.com/help/linkedin/answer/a522427
- 优先级：P0
- 合规方式：发产品故事、使用案例、更新日志，不发硬广海报

### 22. Reddit / r/SideProject
- 类型：社区反馈帖
- 参考讨论：https://www.reddit.com/r/sideprojects/comments/1ld5xve/my_side_project_rsideprojects_new_rules_and_an/
- 优先级：P1
- 合规方式：发 side project 背景、遇到的问题、为什么按场景整理 Skills，比“来用我的网站”更安全

### 23. Reddit / r/SaaS
- 类型：创业 / SaaS 社区
- 合规参考：https://www.business.reddit.com/learning-hub/articles/smb-how-to-use-reddit
- 优先级：P1
- 合规方式：发“我如何验证一个创作者 AI 导航站的需求”或“目录站和社区分发的效果”，不要直接刷链接

### 24. Reddit / r/indiehackers
- 类型：创业者社区
- 规则经验参考：https://www.indiehackers.com/post/free-content-distribution-channels-5108c23bbe
- 优先级：P1
- 合规方式：如果发帖，优先走 SHOW / feedback / learnings 角度，先看当期规则

### 25. X / Twitter 首发线程
- 类型：社交分发
- 优先级：P1
- 合规方式：发 1 条主线程，内容写：
  - 为什么做这个站
  - 它解决什么问题
  - 适合谁
  - 3 个可直接点击的页面

### 26. Product Hunt 产品论坛 / 后续线程
- 类型：产品社区讨论
- 参考：https://www.producthunt.com/posts/interactive-launch-guide
- 优先级：P2
- 合规方式：如果上了 Product Hunt，后续在产品论坛继续更新使用场景、收录标准和新增页面

## 每个平台统一准备的素材

- 网站名称：OpenClaw Skills for Authors
- URL：https://www.clawauthor.com
- 一句话介绍：A curated OpenClaw skills directory for authors and content creators.
- 长描述：按写书、写文章、写文案、写报告、写论文、写课程 6 类场景整理 OpenClaw Skills，并支持按工作环节继续筛选。
- 分类标签：
  - AI Tools
  - Writing
  - Productivity
  - Content Creation
  - Education
  - Research
- Logo
- 站点截图 3-5 张
- 联系邮箱
- 社交链接（LinkedIn / X）

---

## 合规提醒

- 不要同一天群发 20 个一模一样的简介
- 不要在 Reddit / HN / IH 只丢一个链接
- 不要夸大“最好 / 第一 / 最权威”这类无法证明的话
- 每次发社区帖都先看当日规则
- 目录站更偏收录，社区站更偏叙事与案例
